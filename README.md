## Description

This is a simple Organizational structure management system

## Requirement

- Latest version of Node.js
- PostgreSQL dbms

## Configuration

- create .env file in the root folder
- add config entries according to the file sample.env file

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
